# MFA MERN Project

A Multi-Factor Authentication system built with the MERN stack (MongoDB, Express, React, Node.js).

## Features
- User registration and login
- OTP verification via email
- Protected dashboard

##  Prerequisites
- Node.js installed
- MongoDB installed and running
- Email service credentials (for OTP)
